# Python Learning — Day 1

A record of my first day learning Python. Each file builds on the last concept.

## Files

| File | What it teaches | What it does |
|------|-----------------|--------------|
| `pl.py` | Variables, data types, print | Declares variables of different types and prints them with type info |
| `input.py` | Taking user input | Asks for name and age, prints a greeting |
| `Ex1.py` | Using `int()` to convert input, math | Takes birth year, calculates current age (2026) |
| `fav.py` | Lists, loops, indexing | Stores 5 lists of favourite things, indexes, counts, and loops through them |
| `calc.py` | Functions, conditionals | Calculator with 4 operations — user picks operation and enters two numbers |
| `ex2.py` | Conditionals, modulo operator | Takes a number and checks if it's positive/negative/even/odd |

## How to run

```bash
python filename.py
```

Each script will either ask for input or print output directly.

## What I learned

- Variables and data types (str, int, float, bool)
- Taking input with `input()`
- Lists and indexing
- Loops (for, while)
- Conditionals (if, elif, else)
- Functions with `def`
- Type conversion with `int()`
- The modulo operator `%` for remainders
